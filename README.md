# STerm

跨平台的 SSH / SFTP 终端客户端，支持 Windows / macOS / Linux。

## 功能

### 终端
- **SSH 交互式终端**：支持密码与私钥两种认证方式，提供功能完整的终端体验，自动适配窗口尺寸。
- **多会话标签页**：在多个会话间快速切换，互不干扰。
- **终端主题**：内置多套配色方案，可按喜好切换。

### 文件管理
- **SFTP 文件浏览**：远程目录浏览、逐级进入子目录。
- **上传 / 下载**：通过系统文件对话框选择文件，便捷传输。
- **文本在线预览与编辑**：直接查看并编辑远程文本文件，无需先下载。

### 连接与配置
- **连接管理**：保存、编辑常用的 SSH 连接，一键发起连接。
- **主密码保护**：密码、私钥等敏感信息以主密码加密存储在本地。
- **配置同步**：通过 GitHub Gist 在多台设备间同步连接与设置。
